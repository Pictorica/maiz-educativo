# Audio Files for Maiz Educativo Quiz

This directory should contain the following audio files for the quiz:

## Required Audio Files

1. **bg-loop.mp3** - Background music (looping)
   - Should be a calm, educational-themed background track
   - Recommended: 8-bit or chiptune style music
   - Duration: 30-60 seconds (will loop)

2. **correct.wav** - Correct answer sound effect
   - Short, positive sound (e.g., "ding", "success chime")
   - Duration: 0.5-1 second

3. **wrong.wav** - Wrong answer sound effect
   - Short, neutral or gentle "incorrect" sound
   - Duration: 0.5-1 second

4. **finish.mp3** - Quiz completion sound
   - Celebratory sound for finishing the quiz
   - Duration: 2-4 seconds

## File Format Recommendations

- **Background music**: MP3 (good compression, widely supported)
- **Sound effects**: WAV (high quality, instant playback) or MP3
- **Bitrate**: 128-192 kbps for MP3 is sufficient for web use
- **Sample rate**: 44.1 kHz standard
